コミュニティ

Infinispanは、あなたの参加を歓迎するオープンソースの貢献者のグループによって作成され、維持されています。

公開プロジェクト

Infinispanは、Apache Public License 2.0のもと、100%オープンソースのプロジェクトです。コードがオープンソースであり、自由に使用でき、自由に変更できるだけでなく、開発プロセスもオープンです。

コードと課題追跡

ソースコードはGitHubでホストされています。バグを報告したり、新機能を要求する必要がある場合は、JIRA issues trackerで同様のものを探してください。見つからない場合は、新しい課題を作成してください。

StackOverflow

質問がある場合、バグが発生している場合、Infinispanを使用する上でアドバイスが欲しい場合は、StackOverflowをご利用ください。できるだけ早くお答えできるように努力いたします。

礼儀作法

助けを求めているのであれば、まずドキュメントに目を通しましたか？そこに答えがあるかもしれません。

また、バグを報告する場合は、まずバグトラッカーをチェックしましたか？もしかしたら、既知の問題かもしれません。

開発者メーリングリスト

これはInfinispan自体の開発を目的としています。Infinispanが内部でどのように機能しているかについての深い技術的な議論が含まれていることが多いです。もしあなたがInfinispanに貢献しようとしているのであれば、私たちはあなたの考えやアイデア、そして開発者向けのメーリングリストへの参加を歓迎します。

アーカイブ

Infinispanの開発者向けメーリングリストのアーカイブは、優れた検索機能を持つMarkMailで見れます。また、アーカイブされたInfinispan wikiでも古いコンテンツを見つけられます。

TwitterでInfinispanをフォロー

私たちの最新情報は、ほとんどがTwitterで発信されています。プロジェクトチームが業界のイベントで講演することもあれば、人々がInfinispanを使うクールな方法を投稿することもあります。ぜひリツイートしてください。

チャットで相談

Infinispanコミュニティでは、リアルタイムのコミュニケーションにZulipChatを使用しています。InfinispanチャットではWebブラウザや専用アプリケーションを使って参加できます。

公開プロジェクトのロードマップ

コミュニティの今後の計画についてお読みください。これが現在のロードマップです。これはコミュニティによって進められているもので、あなたのニーズを教えてください。

継続的インテグレーション

Infinispanのビルドプロセスを追跡するために、Jenkinsダッシュボードをチェックしてください。

普及活動

Infinispanコミュニティを支援する最もシンプルで簡単な方法は、プロジェクトの大使として活動すること、Twitterでフォローすること、コミュニティのイベントに顔を出すことです。

出版物

Infinispanの興味深い使用例や実例を紹介した親切な記事やブログ記事がたくさんあります。

情報開示

私たちは、このプロジェクトがどのように機能し、誰がどのような責任を負うのかを明確にしたいと考えています。関係者全員が、どのように決定がなされ、どこで貢献できるのかを理解したいと考えています。もし、このページがプロジェクトの所有権と責任の現状を反映していないと思われる場合は、開発者用メーリングリストでプロジェクトリーダに連絡してください。

管理

このプロジェクトは、Java 開発者コミュニティの支援を受けて Red Hat が開発・公開しています。プロジェクトリーダは Red Hat によって任命され、プロジェクトへの貢献を受け入れるか否か、ロードマップを設定する権限を持っています。

リリースサイクル

Infinispanは、メジャー(新機能とAPIの変更)、マイナー(新機能だが壊れるようなAPIの変更はない)、マイクロリリース(パッチ)をリリースしています。InfinispanのあるバージョンがRed Hat Data GridのリリースのGAに含まれると、Infinispanは次のメジャーバージョンに移行します。しかし、Infinispanは定期的にリリースされるため、オープンソースバージョンの作業は常に新鮮で最先端であり続けられます。長期的にパッチを提供してもらいたい場合は、Red Hat Data Gridの使用をお勧めします。

要件定義

Infinispan の要件とロードマップは、コミュニティによって推進されます。通常、メジャーリリースが計画されている場合、プロジェクトリーダーは、Red Hatコミュニティを含むコミュニティの全員からの意見を取り入れ、必要な時間内に何ができるかを評価します。これにより、特定のリリースでどのような機能に対応するかが定義されます。その後、プロジェクトリーダはロードマップをInfinispanの公式なロードマップで公開します。Infinispanの将来のリリースのための要件を提案するための最良の方法は、JIRAに機能要求を入力することです。InfinispanチームはJIRAの投票システムを使用して課題の人気度を評価していますので、コミュニティの他の人にあなたの機能要求に投票するように働きかけてください！

貢献

Infinispanは、コミュニティへの貢献を歓迎し、繁栄させ、大いに評価しており、貢献するためのハードルをできるだけ低くしています。貢献する前に、ドキュメントのInfinispanへの貢献のページを読むことをお勧めします。このページには、プロジェクトに採用される可能性を最大限に高めるコードの書き方のヒントや、Infinispanのコードベースで作業する際に役立つ多くの便利なヒントが含まれています。

Red Hat Data Grid

InfinispanはRed Hat Data Gridのベースとなっているため、Infinispanに対する多くの要求はしばしばRHDGによってもたらされます。これはRed Hat Developersで試せます。
