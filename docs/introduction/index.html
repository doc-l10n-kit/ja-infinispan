Infinispanの紹介

分散型インメモリ・キー/バリュー・データ・グリッドおよびキャッシュ

Infinispanとは？

Infinispanは、非常にスケーラブルで可用性の高いキー/バリュー・データ・ストアおよびデータ・グリッド・プラットフォームです。100%オープンソースで、Javaで書かれています。Infinispanの目的は、分散型、高並列型のデータ構造を公開することであり、最新のマルチプロセッサやマルチコア・アーキテクチャを最大限に活用できるように設計されています。Infinispanは、分散型キャッシュとしてだけでなく、NoSQLのキー/バリュー・ストアやオブジェクト・データベースとしても使用されます。

なぜ使うのか？

ほとんどの人は、Infinispanを2つの理由のどちらかで使っています。第１に、キャッシュとして。データベースやディスクベースのNoSQLストアなど、ボトルネックとなっているシステム部分の前にInfinispanを置くことで、パフォーマンスの向上に大きく貢献します。しかし、単純なキャッシュでは十分ではないことがよくあります。たとえば、アプリケーションがクラスター化されていて、データの一貫性のためにキャッシュの干渉性が重要な場合などです。そのような場合、分散キャッシュが大いに役立ちます。

Infinispanは、高性能なNoSQLデータストアとしても使用できます。メモリ上だけでなく、Infinispanはデータをより恒久的なストアに永続化できます。これをキャッシュストアと呼びます。キャッシュストアはプラグイン式で、簡単に独自のものを実装できますし、すでに多くのものが存在していますので、利用できます。キャッシュストアの詳細と、現在使用可能な既存の実装については、このウェブサイトのキャッシュストアのセクションを参照してください。

もう１つの一般的な使用例は、フレームワークにクラスタビリティと高可用性を追加することです。Infinispanは分散型のデータ構造を公開しているため、クラスター化を必要とするフレームワークやライブラリは、Infinispanを組み込み、すべての状態管理をInfinispanに委ねることで簡単に実現できます。このように、Infinispanにすべてを任せることで、どんなフレームワークも簡単にクラスター化できます。

どこで知ることができますか？

このウェブサイトの ドキュメンテーション セクションをご覧ください。チュートリアル、クイックスタートガイド、サンプルコード、デモなど、多くのリソースが用意されており、すぐに使いこなせるようになります。

どうやって使うの？

コアとなるInfinispanは、java.util.Mapを拡張したCacheインターフェイスを公開しています。また、オプションとして、サーバーのクラスタに効率的にデータを分散させるためのピアツーピア・ネットワーク・アーキテクチャを備えています。

コアとなるJava APIに加えて、InfinispanはHot Rodプロトコルを利用することで、JVM以外のプラットフォームでも利用でき、様々なプラットフォーム向けのクライアントライブラリが存在します。

トランザクションについては？

多くのNoSQLエンジンの間で加熱している話題です。はい、Infinispanは完全なトランザクションに対応しています。InfinispanはJTAとXAの両方の規格をサポートしており、有効なJTAトランザクション・マネージャによって仲介される分散トランザクションに参加できます。

ほとんどの分散データストアは、トランザクションがパフォーマンスを低下させると考えています。これはいくつかのケースでは事実ですが、他の多くのケースでは、トランザクションは多くのビジネスアプリケーションに必要であると考えています。そのため、私たちはトランザクションをサポートしていますが、これはオプションであり、パフォーマンスを向上させるために無効にできます。

Hibernateで使用できますか？

コアとなるInfinispanは、java.util.Mapを拡張したCacheインターフェイスを公開しています。また、オプションとして、サーバーのクラスタに効率的にデータを分散させるためのピアツーピア・ネットワーク・アーキテクチャを備えています。

コアとなるJava APIに加えて、InfinispanはHot Rodプロトコルを利用することで、JVM以外のプラットフォームでも利用でき、様々なプラットフォーム向けのクライアントライブラリが存在します。

検索はできますか？

はい、Infinispan の主なデータ検索方法は、キーによる検索 ( Cache.get(key) ) ですが、データセットに対する強力な索引作成および検索もサポートしています。詳細は、ドキュメントをご覧ください。

Map/Reduceについては？

さらに素晴らしいことに Infinispanでは、Javaの非常に強力なStream APIを使用することができ、ローカルおよびクラスター化されたすべてのモードで、データに対してローカルにコードを実行できます。Streamに加えて、Infinispanは分散コード実行もサポートしており、処理をグリッドに移せます。

標準

JSR 107

JSR 107 (JCACHE: Javaのための一時的なキャッシング)は、Infinispanの開発チームが積極的に参加している規格で、エキスパート・グループの一員でもあります。Infinispanは、JSR 107のAPIが最終的に決定され次第、実装する予定です。JSR 107 APIのプレビューについては、GitHubのJSR 107をご覧いただくか、InfinispanのJSR 107モジュールをご覧ください。
