Infinispan 11.0.1.Final "Corona Extra"

July 03, 2020 Tags: release

By Tristan Tarrant

親愛なるInfinispanコミュニティへ

最新のメジャーリリースである Infinispan 11 に含まれるすべての新機能をお楽しみいただけましたでしょうか。私たちがみなさんを大切にしていることを示すために、いくつかの問題を解決するための新しいマイクロリリースをご用意しました。

特に、TLS/SSLでHTTP/2を使用している場合、JCacheと永続化、Spring Boot、またはRocksDBを使用している場合は、我々はあなたのための修正を用意しています。

さらに、Infinispan のアーキタイプは復活し、メインリポジトリの一部としてメンテナンスされており、これ以上同期が取れなくなることはありません。 Mavenのアーキタイプを使い始める 方法の詳細を読んでください。

現在、新機能、改善、修正に取り組んでいますので、今後の発表にご期待ください。

最新のリリースをダウンロードしてお試しください。

ソースコードはGitHubでホストされています。 バグを報告したり、新機能を要求したりする必要がある場合は、JIRA課題監視で同様のものを探してください。 見つからない場合は、新しい課題を作成してください。

質問があったり、バグが発生していたり、Infinispanを使う上でのアドバイスが欲しい場合は、 StackOverflowをご利用ください。 できる限り早く回答するように努めます。

Infinispanのコミュニティでは、リアルタイムのコミュニケーションにZulipを使用しています。 WebブラウザまたはInfinispanチャット上の 専用のアプリケーションを使って参加してください。
