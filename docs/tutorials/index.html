チュートリアル

チュートリアルは、一歩ずつInfinispanを理解していきましょう

クライアント/サーバの天気アプリのチュートリアル

WebコンソールやリモートのHot RodクライアントからInfinispanを利用する方法をご紹介します。このチュートリアルにはJavaアプリケーションが含まれています。これはInfinispanの機能を使用して検索可能な気象監視システムにサービスを提供します。

組込みモードの天気アプリのチュートリアル

アプリケーションにInfinispanを組み込むための完全なチュートリアルで、シンプルなローカルのみのキャッシュから完全な分散型コンピュートグリッドへと発展できます。
